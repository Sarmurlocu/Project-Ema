# Project-Ema (東方支持系統)

這是一個以觀音一百籤為核心，結合 AI 心理支持濾鏡的導引系統。

## 核心設計規範 (Filtering Conditions)
1. **Industry Cluster (產業聚落)**：識別使用者焦慮來源。
2. **Risk (風險)**：溫暖地提示潛在風險，不使用恐嚇語氣。
3. **VCP (價值創造)**：引導使用者發現自身當下的價值。

## 數位憲法 (Ethics & Tone)
- **虛化主詞**：不使用「我、你、孩子」，維持靜謐的古老智慧感。
- **去等級化**：移除上上/下下標籤，將能量轉化為純粹的支持文字。
- **緊急保護**：當識別到極端焦慮時，導向「生命韌性保護」專區。
